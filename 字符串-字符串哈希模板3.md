#### 力扣 214 最短回文串
#### 给定一个字符串 s，你可以通过在字符串前面添加字符将其转换为回文串。找到并返回可以用这种方式转换的最短回文串。

要在字符串 s 前面添加最少的字符使其变为回文串，本质上是要在 s 中找到 最长的回文前缀。

假设最长回文前缀的长度为 best，那么我们只需要将 s 剩余的部分（从 best 到末尾）翻转过来，拼接在 s 的最前面即可。

字符串哈希的逻辑判断一个字符串是否为回文，标准做法是看它 正着读 和 反着读 是否相同。

在滚动哈希中，我们可以同时维护两个哈希值：

正向哈希 ($h_1$)：计算前缀 $s[0 \dots i]$ 的哈希值。

反向哈希 ($h_2$)：计算前缀 $s[0 \dots i]$ 翻转后的哈希值。

如果 $h_1 == h_2$，说明 $s[0 \dots i]$ 极大概率是一个回文串。

注意：反向更新时，新加入的字符 $s[i]$ 其实变成了反转后的“最高位”，所以它要乘以 $\text{base}^i$。

#### 要重点理解并掌握滚动哈希的两个公式!!!

代码：

```cpp
class Solution {
public:
    string shortestPalindrome(string s) {
        int n = s.size();

        if (n == 0)  return "";

        long long base = 131;
        long long mod = 1e9 + 7;
        long long leftHash = 0, rightHash = 0;
        long long pow = 1;
        int best = 0;

        for (int i = 0; i < n; i ++ ) {
            leftHash = (leftHash * base + s[i]) % mod;

            rightHash = (rightHash + pow * (long long)s[i]) % mod;
            pow = (pow * base) % mod;

            if (leftHash == rightHash) {
                best = i + 1;
            }
        }

        string remain = s.substr(best);
        string rev_remain = remain;
        reverse(rev_remain.begin(),rev_remain.end());

        return rev_remain + s;
    }
};
```
