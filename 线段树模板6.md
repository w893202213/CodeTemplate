#### 洛谷P4145 势能线段树经典题
#### 本题需要动态求区间和 + 对区间开根号

开方操作不满足分配律（即 $\sqrt{a+b} \neq \sqrt{a} + \sqrt{b}$），因此无法像加法或乘法那样通过简单的“懒标记（Lazy Tag）”来统一更新整个区间。

但是，这道题有一个非常核心的数学特性，可以让我们通过“暴力”手段解决。

### 核心思路：势能分析与暴力更新
1. 数值的收敛性题目给出的数据范围中，数列中的数不超过 $10^{12}$。

   请注意一个神奇的现象：

   $\sqrt{10^{12}} = 10^6$

   $\sqrt{10^6} = 1000$

   $\sqrt{1000} \approx 31$

   $\sqrt{31} \approx 5$

   $\sqrt{5} \approx 2$

   $\sqrt{2} \approx 1$

   $\sqrt{1} = 1$

   这意味着，任何一个在 $10^{12}$ 以内的正整数，经过最多 6 到 7 次开方操作后，都会变成 1。而一旦数字变成了 1，再次开方的值依然是 1，不再发生变化。

2. 区间剪枝（关键优化）虽然我们不能用懒标记，但我们可以给线段树的每个节点维护一个最大值 max_val：

在执行区间开方操作时，如果当前节点覆盖的范围 $[l, r]$ 内，其 max_val <= 1，说明这个区间里的所有数已经是 1（或 0），开方操作对它们无效。此时，我们直接 Return，不再向下递归。

3. 为什么不会超时？
  
   这种做法看起来是“暴力”修改叶子节点，但实际上：每个原始节点最多只能被有效更新 6-7 次就会变成 1。

   一旦一个区间全是 1，后续的修改操作都会被 max_val <= 1 这个条件直接拦截。

   整个算法的总复杂度大约是 $O(7 \times n \log n + m \log n)$，在 $10^5$ 的数据量下完全跑得动。

#### 着重关注 modify 函数中的递归出口和剪枝的逻辑

代码：

```cpp
#include <iostream>
#include <cstdio>
#include <cstring>
#include <algorithm>
#include <cmath>

using namespace std;

typedef long long LL;

const int N = 1e5 + 10;

int n, q;

LL arr[N];
LL sum[N << 2], maxv[N << 2];

void up(int u) {
    sum[u] = sum[u << 1] + sum[u << 1 | 1];
    maxv[u] = max(maxv[u << 1], maxv[u << 1 | 1]);
}

void build(int l, int r, int u) {
    if (l == r)
        sum[u] = maxv[u] = arr[l];
    else {
        int mid = l + r >> 1;
        
        build(l, mid, u << 1);
        build(mid + 1, r, u << 1 | 1);
        
        up(u);
    }
}

void modify(int jobl, int jobr, int l, int r, int u) {
    // 注意这里不是 jobl <= l && r <= jobr, 因为必须要递归到最底层的单个点
    if (l == r) {
        LL num = LL(sqrt(maxv[u]));
        maxv[u] = sum[u] = num;
    } else {
        int mid = l + r >> 1;
        
        // 核心剪枝：maxv 大于 1 时才有继续开根号的必要
        if (jobl <= mid && maxv[u << 1] > 1)
            modify(jobl, jobr, l, mid, u << 1);
        if (mid + 1 <= jobr && maxv[u << 1 | 1] > 1)
            modify(jobl, jobr, mid + 1, r, u << 1 | 1);
        
        up(u);
    }
}

LL query(int jobl, int jobr, int l, int r, int u) {
    if (jobl <= l && r <= jobr)
        return sum[u];
    else {
        int mid = l + r >> 1;
        
        LL res = 0;
        if (jobl <= mid)
            res += query(jobl, jobr, l, mid, u << 1);
        if (mid + 1 <= jobr)
            res += query(jobl, jobr, mid + 1, r, u << 1 | 1);
        
        return res;
    }
}

/*
    势能线段树经典题
*/
int main() {
    cin >> n;
    
    for (int i = 1; i <= n; i ++ )
        scanf("%lld", &arr[i]);
    
    build(1, n, 1);
    
    cin >> q;
    
    int op, l, r;
    while (q -- ) {
        scanf("%d%d%d", &op, &l, &r);
        
        if (l > r)  swap(l, r);
        
        if (op == 0)
            modify(l, r, 1, n, 1);
        else
            printf("%lld\n", query(l, r, 1, n, 1));
    }
    
    return 0;
}
```
